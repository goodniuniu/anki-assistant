# 数据目录说明

本目录包含项目的数据文件。

## 目录结构

```
data/
├── input/          # 输入数据（原始数据）
│   ├── raw/       # 原始文本文件
│   └── extracted_120/  # 从 Anki 卡包提取的数据
├── cleaned/        # 清洗后的数据
└── output/         # AI 增强后的输出文件
```

## 输入格式要求

### txt 格式（推荐）

使用 Tab 分隔的两列格式：

```
正面文本1	背面文本1
正面文本2	背面文本2
```

### csv 格式

```csv
Front,Back
正面文本1,背面文本1
正面文本2,背面文本2
```

### xlsx 格式

Excel 文件，包含两列：
- 第一列：Front（正面）
- 第二列：Back（背面）

## 注意事项

- **编码**：所有文件请使用 UTF-8 编码
- **分隔符**：txt 文件使用 Tab（`\t`）分隔
- **换行**：使用 Unix 风格的换行符（`\n`）
