
### 2. 导入 Anki
1. 打开 Anki
2. 文件 → 导入
3. 选择 `ancient_words_793_enhanced.txt`
4. 模型选择: "基础（及反面反转）"
5. 字段映射:
   - Column 1 → Front (正面)
   - Column 2 → Back (背面)
6. 编码选择: UTF-8
7. 点击"导入"

### 3. 开始复习
- **正面**: 包含关键词语的古文句子
- **背面**: 详细的词语解释、出处、用法、例句
